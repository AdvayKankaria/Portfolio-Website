# Advay Kankaria — Portfolio

A high-performance, design-engineered personal portfolio website built for speed, aesthetics, and interaction. Featuring a fully responsive bento-grid layout, glassmorphic UI elements, and bespoke WebGL (Three.js) animations.

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- **3D Graphics**: [Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/)
- **Typography**: Geist Sans & Geist Mono
- **Deployment**: Vercel

## Key Features

- **Agentic AI & Security Focus**: Showcasing projects across Machine Learning, Federated Learning, and Cyber Security.
- **Glassmorphic UI**: Premium frosted glass effects, glowing interactive elements, and staggered scroll-reveal animations.
- **WebGL Hero**: Custom, lightweight 3D floating geometries interacting dynamically with a fluid shader background.
- **Dynamic Grid Layouts**: Asymmetrical masonry grids for projects and responsive bento-box layouts.
- **Performance Optimized**: Reduced motion hooks, lazy-loaded Canvas elements, and static generation.

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/AdvayKankaria/Portfolio-Website.git
   cd Portfolio-Website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is fully optimized for zero-config deployment on Vercel. 
Simply connect the repository to Vercel and it will automatically build using `next build` and deploy.
